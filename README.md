### Manual
+ **Soft-keys** Zoom the map
+ **Cursor Moving** the map
+ **key 1** update Geolocation-> Marker
+ **key 2** search
+ **key 3** load geoJson Track
+ **key 4** show Position 

### GeoJson
+ Online tool: [geojson.io](http://geojson.io/#map=1/-55/228)
+ Validation: [geojsonlint](http://geojsonlint.com/)
