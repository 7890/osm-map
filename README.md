This is an Open Street Map app for the Nokia 88104G - [KaiOS](https://www.kaiostech.com/) 


### Manual
+ **Soft-keys** Zoom the map
+ **Cursor Moving** the map
+ **key Enter** Center the map
+ **key 1** update Geolocation-> Marker
+ **key 2** search
+ **key 3** show Position
